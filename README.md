# vaccine_app
설명
인도의 백신 접종자수와 백신 접종 부작용에 관한 데이터를 분석한 웹대시보드입니다.

백신 접종자의 종합적인 최대/최소 데이터를 알수 있으며, 각 컬럼별 기준으로 히스토그램과 파이차트를 확인할수 있습니다.

마지막으로 접종자수를 입력하면 인공지능으로 부작용수를 예측하여 보여줄수 있습니다.
